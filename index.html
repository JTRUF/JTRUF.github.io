Projects

The Christmas Tree Lady

Visit the site: ashleethechristmastreelady.com

Visit the GitHub repository: The Christmas Tree Lady GitHub

A website designed for a business that sells custom designed Christmas tree kits. I used React, JavaScript ES6, HTML5, CSS3, Node, PostgreSQL, Express, and Massive to build this project. I integrated Auth0 to allow users to login to the website, used PostgreSQL to manage each product on the Products page, used Amazon S3 to store the images for the Product's page, utilized NodeMailer on the Contact page so the user can send a message directly to the Admin's email address, implemented Stripe on the Cart page to manage all secure payments when the user wants to checkout, and programmed Media Queries which allows the web application to be fully mobile responsive.

CryptoKitties Clone

Visit the site: CryptoKitties Website

Visit the GitHub repository: CryptoKitties GitHub

As a group, we cloned the CryptoKitties website. CryptoKitties is a blockchain based virtual game that allows players to purchase, collect, breed and sell various types of virtual cats. It is one of the earliest attempts to deploy blockchain technology for recreational and leisurely purposes. CryptoKitties is not a cryptocurrency. Instead it operates on Ethereum's underlying blockchain network, as a non fungible token (NFT) unique to each CryptoKitty.
